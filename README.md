# Data Warehouse & Analytics Project

A portfolio project showcasing a modern data warehouse built on **Medallion Architecture** with Bronze, Silver, and Gold layers.

## What it covers
- **Data Engineering**: Ingesting raw CSV data from ERP and CRM source systems into a SQL Server database
- **ETL Pipelines**: Cleaning, transforming, and standardizing data across layers
- **Data Modeling**: Building a star schema with fact and dimension tables optimized for analytics
- **Analytics & Reporting**: SQL-based reports delivering insights on customer behavior, product performance, and sales trends

## Tech Stack
PostgreSQL, pgADMIN4, SSMS, Git, DrawIO

## Architecture
| Layer | Purpose |
|-------|---------|
| Bronze | Raw data as-is from source systems |
| Silver | Cleansed and standardized data |
| Gold | Business-ready data for reporting |
